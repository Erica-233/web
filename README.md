# Web项目

这是一个包含多个HTML页面的静态网站项目。

## 项目结构

- `555.htm` - 主页面
- `555-1.htm`, `555-2.htm`, `555-3.htm` - 子页面
- `news.htm` - 新闻页面
- `video.htm` - 视频页面
- `search.htm` - 搜索页面
- 各种图片文件 (`.jpg`)

## 使用方法

直接在浏览器中打开任意HTML文件即可查看网站内容。
